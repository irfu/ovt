Release notes and system requirements for OVT version 3.0

April 19, 2016

New Features

  • Now works on Mac OS X, Windows, and Linux, with identical functionality.
  • OVT is installed using an installer software. There is no need to separately install VTK or Java (except for Java which needs to be preinstalled on Linux).
  • Ability to use online orbit data from NASA SSC.
  • Ability to use online/offline NASA OMNI2 solar wind and magnetospheric data.

System requirements

Linux:

Requires 64-bit Linux with OpenGL support. Requires an already installed version of Java Standard Edition 8. Tested successfully on openSUSE 13.1, and Ubuntu 14.04.

Mac

Successfully tested on Mac OS X 10.09+.

Windows:

Successfully tested on Windows 7 (64-bit).
