Reporting bugs:
Bugs should be reported by email to: ovt-team@irfu.se
Please include the exact version number of OVT, the operating system, and a detailed description of the bug and how to reproduce it. Terminal output or logs are also helpful. Logs can be found under the application directory and/or under .ovt/<version number>/ directory under the user directory. They are identifed by the suffix ".log".
Note: Before you submit, please check the OVT bugs/issues page (https://bitbucket.org/ovtirfu/ovt/issues) if the bug has been submitted before.
